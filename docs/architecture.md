
## Architecture Overview

This project migrates a 5-tier vProfile application from on-prem to AWS
using Lift & Shift strategy.

### Components
- Nginx – Reverse proxy
- Tomcat – Application server
- MySQL – Database
- Memcached – Cache
- RabbitMQ – Message broker

### AWS Services Used
- EC2
- AMI
- Application Load Balancer
- Auto Scaling Group
- Security Groups
