## Migration Steps

1. Deployed individual EC2 instances for each component
2. Configured MySQL and imported database
3. Connected Tomcat with DB, Memcached, RabbitMQ
4. Verified application on single EC2
5. Installed Nginx as reverse proxy
6. Created AMI from application EC2
7. Created Launch Template from AMI
8. Created Auto Scaling Group
9. Integrated Application Load Balancer
10. Performed instance refresh
11. Validated application via ALB DNS
